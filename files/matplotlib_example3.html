Asset Backed Securities Ratings Over Time¶

Was curious to see how those securities initially rated AAA fared over time - a fair number were downgraded during the crisis, but did they stay that way? Probably not, given the rise in upgrades after the crisis. While probably home-equity/subprime/res BC (pick whatever you want to call the category) were most affected, wanted to focus just on consumer ABS first.

Some rating coverage statistics:¶

Some caveats: I don't include those ratings that the rating agencies actively decide not to rate as being rated. 'No rating' is the same as 'unrated'. There are a tiny proportion of securities among the 'rated' category that are technically Withdrawn ratings, but since rating agencies usually withdraw ratings before payoff, the overall amounts impacted are quite small. These figures also don't include notional/io/exchangeable tranches, either.

When doing this analysis, I was rather surprised that S&P edged out over Moody's in rated securities. It's quite close, though.

# set up the second chart
fig2, ax2 = plt.subplots(figsize=(5,5))
collateral_choice = 'ABS - Auto'

def make_chart(c_t, q_e):
    # some global variables since hadn't planned that far in the ntoebook
    global fig2
    global ax2
    global plt
    global outstandingAAA
    
    # fixed colors palette - just to keep the ratings consistent colors
    colors = {'': '#dbe9f6', '01. Aaa\AAA': '#dbe9f6', '02. Aa\AA': '#bad6eb',
          '03. A\A': '#89bedc', '04. Baa\BBB and Below': '#539ecd', '05. NR': '#2b7bba', '99. Paid Off': '#0b559f'}
    
    collateral_type = c_t
    quarter_end = q_e
    
    # get all of the specific collateral type at the specific quarter
    subclass_issue = outstandingAAA[(outstandingAAA['CollateralType'] == collateral_type) & 
                                (outstandingAAA['OutstandingQuarter'] == quarter_end)]
    x_axis = ['Original AAA', quarter_end]

    # list all the ratings the AAA has been downgraded to or stayed unchanged 
    ratings = subclass_issue['OutstandingRatingBucket2'].tolist()
    ratings.append('99. Paid Off')
    ratings.insert(0, '')

    ratings_outstanding = subclass_issue['OutstandingUSDMillions'].tolist()
    paid_off = subclass_issue['IssuanceUSDMillions'].sum() - subclass_issue['OutstandingUSDMillions'].sum()
    ratings_outstanding.append(paid_off)
    ratings_outstanding.insert(0, 0) # this accounts for original amount at issuance
    ratings_issuance = []
    ratings_issuance.append(subclass_issue['IssuanceUSDMillions'].sum())

    # append 0s to deal with left stacked bar without any non AAA ratings
    [ratings_issuance.append(0) for x in range(len(ratings_outstanding)-1)]

    # build the charting elements
    ratings_io = list(zip(ratings_issuance,ratings_outstanding))

    ax2.set_title('{}: AAA ({})'.format(collateral_type, quarter_end))
    ax2.set_ylabel('$ millions')
    for i in range(len(ratings)):
        if i == 0:
            plt.bar(x_axis, ratings_io[i], label = ratings[i], color = colors[ratings[i]])
            stacked_sum = np.array(ratings_io[0])
        else:
            plt.bar(x_axis, ratings_io[i], label = ratings[i], bottom = stacked_sum, color = colors[ratings[i]])
            stacked_sum += np.array(ratings_io[i]) 
    ax2.legend(loc='lower left')
    plt.draw()

chart_counter = 0
quarters = outstandingAAA.OutstandingQuarter.unique()
make_chart(collateral_choice, '2000Q1')
plt.show()
    
def onclickdown(event):
    global chart_counter
    global quarters
    
    if chart_counter == len(quarters)-1:
        chart_counter = 0
    else:
        chart_counter += 1
    ax2.cla()
    make_chart(collateral_choice, quarters[chart_counter])
    
# on click down, the chart updates and iterates through to the next quarter. 
fig2.canvas.mpl_connect('button_press_event', onclickdown)
